# Graph Enrichment Data Layout

This repository keeps a human-readable reference layout for the Graph Enrichment Memory System.

Runtime writes happen under Electron user data at:

- `userData/data/local/`
- `userData/data/exports/json/`
- `userData/data/exports/csv/`
- `userData/data/snapshots/`
- `userData/data/schema/`
- `userData/data/migrations/`
- `userData/data/cloud/`
- `userData/logs/enrichment/`
- `userData/logs/validation/`

This `data/` tree documents the expected structure and cloud-ready contracts.
